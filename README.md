## 📋 개요

게임 개발자로서의 제 기술, 프로젝트, 경험을 보여주는 개인 이력서 웹사이트입니다. 웹사이트는 다음 기술들로 구성되어 있습니다:

- HTML5
- Tailwind CSS
- Font Awesome Icons
- Google Fonts (Poppins)

## 🚀 주요 기능

- 모든 기기에서 작동하는 반응형 디자인
- 한국어/영어 이중 언어 지원
- 현대적이고 깔끔한 UI
- 인터랙티브 프로젝트 카드
- 부드러운 애니메이션
